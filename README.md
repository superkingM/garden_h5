# 魔法花园

魔法花园是QQ家园中的一款以种植花朵养成类的文字游戏，随着QQ家园的关闭，这款游戏也消失在了历史的长河中。

这里凭借自己记忆中的样子和资料对这款游戏进行玩法复刻，将以前的老式页面UI升级为现代移动端UI。

## 玩法功能
- [x] 花园:播种、收割、操作
- [x] 魔法屋:花种合成
- [ ] 花房:点亮的花朵
- [ ] 好友:好友间操作
- [ ] 活动:参与获取奖励
- [x] 背包:查看背包中的花种与道具
- [x] 商店:购买花种与道具
- [x] 排行榜:花谱榜、等级榜、财富榜
- [x] 签到:获取花种
- [ ] 帮助：指引一些玩法
